# WEBSITE LAVANDERIA - PROJETO UC9

Este é o repositório oficial do website que simula uma empresa de lavanderia, que visa ser um site responsivo, e que deve ser apresentado aos devidos professores, para a conclusão satisfatória da unidade curricular nº9 (UC9) do curso de Desenvolvimento de Sistemas (DS).

Atual estado: **em desenvolvimento**
